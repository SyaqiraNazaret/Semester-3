position: static;

Position static digunakan untuk mengatur element menjadi statis scara default. Elemen akan mengikuti posisi normal secara default, elemen tidak dipengaruhi oleh properti seperti top, bottom, left dan right.:

Elemen diatur menggunakan posisi: static;




position: relative

Sebuah element HTML yang menggunakan position relative akan terletak pada posisi normal. Mengatur properti atas, kanan, bawah, kiri dari elemen yang diposisikan dengan "relatif" akan membuatnya jauh dari posisi normalnya. Konten lain akan disesuaikan dengan celah yang ditinggalkan oleh elemen tersebut.:

Elemen diatur menggunakan posisi : relative;















position: fixed;

Element HTML yang menggunakan position absolute akan diposisikan relatif dengan element lain yang mendahuluinya yang terdekat, bukan relatif terhadap layar secara normal.:

Elemen ini diatur menggunakan posisi relative:
Elemen ini diatur menggunakan posisi: absolute
